# 2026 관광데이터 분석 포스터 공모전 — 보고서 골격 (v0.1)

> **주제(가제):** 「데이터로 진단하는 지역 관광쇠퇴 — 동선 네트워크와 복합쇠퇴지수로 찾는 활성화 우선지역, 그리고 문화유산·한류 연계 처방」
> **필수데이터:** 국민여행조사(국내), 관광 빅데이터 지역별 방문자수 API, (외래관광객조사 보조)
> **무게중심:** 진단(분석) 7 : 정책제언 3 — 데이터 분석 공모전 성격상 분석 충실성 우선(재검토 필요 시 조정)

---

## 1. 서론 (1매)
- **문제제기:** 국내 관광이 수도권·제주·강원 등 소수 허브에 편중. 방문량 자체보다 "동선에서 소외되고, 매력이 약화되고, 방문이 감소하는" 지역이 진짜 위기.
- **지방소멸과의 연결:** 2020년 수도권 인구가 전체의 50%를 돌파한 초집중 구조에서, "지역소멸의 대응 방안으로 '지역관광진흥'이 현실적 대안으로 떠오르며 … 관광형 관계인구를 확대하여 지역에 활력을 부여"하는 흐름(강지수·최경은, 2022; 권태일·장유진, 2023). → 관광쇠퇴 진단은 곧 지방소멸 대응의 출발점. [투어고인사이트 2023-6호]
- 기존 지역관광 논의는 '방문객 수' 단일지표 중심 → 다차원 진단 부재.
- **연구질문:** ① 관광객은 어느 지역을 '함께·순서대로' 방문하는가(동선) ② 어느 지역이 복합적으로 쇠퇴하는가 ③ 어디를 어떻게 살릴 것인가.
- 인용(가산점): 「스크린관광의 지속가능한 발전」[투어고인사이트 2026-01호], 「K-콘텐츠가 이끄는 인바운드 관광」[2025-4호].

## 2. 데이터 및 분석방법 (1매)
- **데이터:** 국민여행조사 2025 국내여행 원자료(52,185명, 방문지 시군구 231코드, 여행차6×방문지17 = 방문 순서 내장) / 한국관광공사 지역별 방문자수 API(외지인, 2019·2022·2024) / 외래관광객조사(보조).
- **동선 네트워크 방법:** 방문지 순서 `A→B→C`를 **방향엣지**로, 지역=**노드**. 표본가중치(WT_DOM)로 전국 복원. degree/betweenness 중심성으로 허브·소외 측정. (노드-엣지 도식 삽입)
- **복합 관광쇠퇴지수:** 3축 표준화(z) 등가중.
- ※ 코드체계: 방문자수API(행정표준코드) ↔ 설문(설문코드) → 시도명+시군구명 매칭(190/191).

## 3. 동선 네트워크 분석 (1.5매)  ★독창성 핵심
- **[그림] 시도 동선 네트워크 지도** (`관광동선네트워크_시도.png`): 노드=방문량, 색=연결중심성, 화살표=이동흐름.
- **핵심 인사이트:** 방문량 순위 ≠ 연결중심성 순위.
  - **제주:** 방문 최대급이나 연결중심성 최하(0.312) → 동선상 고립(섬).
  - **충남:** 방문 중위이나 연결중심성 1위 → 경유 허브.
  - → "인기 = 동선 중심"이 아님을 정량 입증(네트워크축이 방문량과 비중복).
- 이동흐름 Top: 강원↔경기, 전남↔경남, 경기↔서울(지리적 정합).
- **[그림] 시군구 네트워크**(`_시군구.png`): 허브=전주·종로·포항·해운대·춘천·여수·속초·강릉 / 소외후보=고흥·거창·옥천·강화.
- 참고: 국내여행 62%가 단일방문 → '스쳐가는' 구조.

## 4. 복합 관광쇠퇴지수 설계 (1.5매)  ★충실성 핵심
- **3축(모두 높을수록 쇠퇴):**
  1. **방문쇠퇴** = z(−회복률24/19) + z(−최근추세24/22)  [방문자수 API]
  2. **매력쇠퇴** = z(−재방문의향 A10) + z(−추천의향 A11)  [여행1차 귀속]
  3. **연결소외** = z(−연결중심성)  [동선 네트워크]
  - 지수 = 3축 등가중 평균. (가중치 민감도분석 1문단)
- **★방법론 서사(엄밀성 어필):** 초기 '무숙박률(당일방문)'을 경유지화 지표로 썼으나 → **대도시 근교 입지와 교란**(의왕·과천 등 오탐) 발견 → **재방문·추천 의향(매력)으로 교체** → 검증(재방문 최고=강진·영암·고흥·장흥 남해안, 최저=진천·괴산) → 방문추세(API) 결합 → **다축 정렬**로 견고성 확보.
- 모집단: 도 지역 시·군 145개(광역시 자치구 45개 제외). 활성화 **타깃 선정**은 인구 50만 미만 시·군으로 한정(지방자치법상 대도시 기준) — 근거는 크기가 아니라 ① 지방 관광 활성화의 정책 범위 ② 대도시 방문자수의 비관광 통행 오염(무숙박률 교란과 동일 논리). 대도시(고양·성남·시흥 등)는 지도에 '대도시권 참고'로 표시. (한계: 대도시 진성 매력쇠퇴 신호 일부 손실 가능.)

## 5. 진단 결과 (1.5매)  ★우수성 핵심
- **[그림] 진단 매트릭스 v3**(`관광쇠퇴진단_매트릭스_v3.png`): X=방문규모, Y=복합쇠퇴지수.
- **[그림] 쇠퇴 × 문화유산 잠재력**(`쇠퇴x문화유산_매트릭스.png`): 정책 타깃팅용. 우상단=쇠퇴+문화유산 풍부=1순위.
- **가장 견고한 타깃(3축 모두 쇠퇴):** ⭐**울릉군**(방문↓+매력↓+연결↓, 섬 접근성) · 청송군 · 의정부시.
- **★타깃 2유형(자원 기반 처방 근거):**
  - **문화유산형**(쇠퇴+문화유산 풍부): 상주(112)·창녕(125)·공주(165)·진주(274)·익산(107)·괴산(75)·청주(210)·여주·거창·서귀포(162)
  - **자연·힐링형**(쇠퇴+문화유산 적음): 울릉·청송 → 문화유산 외 전략
- (선택) half-page 인바운드 진단: 외국인 서울·부산·제주 극편중 → 분산 필요(외래관광객조사).

## 6. 정책제언 (2매) — 타깃 유형별 처방  ★활용성
- **진단→처방 직결:** ③연결소외가 진단지표이자 처방(루트연결) 대상. 문화유산 밀도(국가유산 API)로 진단-정책 데이터 연결.
- **① 문화유산 연계형**(상주·창녕·공주·익산·괴산): 국가유산 **목록·상세(좌표) API**로 유산 루트 설계 + **나레이션 API(한/영/일/중)** 다국어 오디오가이드 → map-3 시각화. 콘텐츠(촬영지)는 방문을 폭발시키지만(영월 청령포 입장객 +640%, 단종장릉 +752%, 2026 1분기) 라이프사이클이 짧아 **'세트장이 아니라 지역이 평소 가꿔 둔 문화유산 자산' 위에 얹어 체류형으로 설계**해야 지속된다(정란수, 2026, 스크린관광 2026-01호). [인용: 스크린관광 2026-01호]
- **② 자연·힐링형**(울릉·청송): 접근성·체류 콘텐츠 중심(문화유산 외). **미식**이 유력 — 국내여행자의 62%가 여행지에서 음식관광활동을 하며(자연·풍경감상 81.3% 다음 2위) 22%는 음식이 방문 이유(2021 국민여행조사). 서천 한산 소곡주 양조장 클러스터(2km 70여 개) 순회 투어처럼 **향토식·특산주 순회 체험으로 체류·재방문 유도**. [인용: 미식관광 2022-7호]
- **③ 참여형 루트공유 + 네비인앱**: 인플루언서/이용자 루트 공유 → 데이터 검증 루트 정규 등록(연례 콘테스트) → TMAP 콜라보 인앱 내비(map-3 프로토타입).
- **④ 인바운드·한류 레버**(공통): 외래조사 K-콘텐츠·유적지 활동(Q8a08/05) + **국가유산 행사 API(야행·생생국가유산)** → 소외 타깃에 외국인 유입. 방한 관심계기 1위가 한류 콘텐츠(38.3%)이고 '적극 한류소비 집단'은 재방문·추천 의향이 높으나 여전히 서울 92.8% 집중 → "지방 도시에 K-콘텐츠 연계 체험·촬영지·팬덤 이벤트 확대"가 정책 방향(백처라, 2025). 단 **단순 촬영지 인증샷은 한계**이므로 "그 지역에서만 체험 가능한 K-콘텐츠 상품"으로 설계(권태일·장유진, 2023). 지방 유입 여지는 실측됨(외국인 경상권 방문 9.6%→12.1%, 2023→2024). [인용: K-콘텐츠 인바운드 2025-4호 + 한류 K-POP 2023-7호]
- 기대효과: 허브 낙수 → 소외지 연계 → 체류·재방문 제고.
- **★해외 벤치마킹(정당성 보강):** 주요 관광국의 국가계획도 '소수 허브 편중 완화 = 지방 분산·질적 전환'을 핵심 축으로 삼는다.
  - **일본 「제5차 관광입국추진기본계획」**(2026): "인바운드 급증 부작용(오버투어리즘) 해결을 최우선 과제로 … 양적 확대에서 질적 향상으로 정책 방향 전환", "관광객의 지방 유치 … 지역 순회-장기체류-소비 확대 촉진". KPI를 방문객 수에서 **1인 소비 25만 엔·재방문객 4천만 명·주민 삶의 질 조화 지역 100개**로 재설정. [국제관광동향 2026-3호, pp.5–9]
  - **스페인 「2030 관광계획」**: "관광객의 **지리적 분산 촉진**", "2차 관광지를 홍보하며 방문객 수의 무분별한 증가를 방지", "인구감소에 직면한 지역 … 농촌관광 활성화". [국제관광동향 2026-2호, pp.5–10]
  - **영국 방문자경제 전략**: 핵심을 "**런던을 넘어선 관광산업의 성장**"으로 규정(방문객 57%가 "런던 외 지역을 모른다"), 지역 방문자경제 파트너십(LVEP) 35개로 지방 분산 제도화. [국제관광동향 2025-1호, pp.5–8]
  - **영국 재생관광**: "지역을 유지·지속하는 것을 넘어 **실질적 변화를 지향**", 리버풀 유로비전 재방문 5.5만 명·경제효과 1,100만 파운드 등 쇠퇴도시 재생 실증. [국제관광동향 2025-7호, pp.5–8]
  - → 본 연구의 '소외지 동선 연계·체류 제고' 처방은 한국만의 임기응변이 아니라 **국제 관광정책의 공통 방향(수도 집중 완화·지방 분산·질적 전환)과 정합**한다.
