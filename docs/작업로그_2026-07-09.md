- 모바일(≤520px)에선 태그라인 숨김.
- CSS: `.brand-txt / .brand-ko / .brand-en / .brand-tag`.

### 1-2. 한국관광공사 출처 배지 (우상단)
- **공식 로고 PNG** `assets/kto_logo.png` (Wikimedia Commons `File:Korea-Tourism-Organization-ko.svg`를 960×318 PNG로 렌더).
- 로고 로드 실패 시 이모블럼+텍스트로 **자동 폴백**(`onerror`).
- **클릭 시 knto.or.kr 새 창** + hover 강조.
- 구성: `[로고] │ TourAPI · 데이터 제공` (로고에 이미 "한국관광공사" 워드마크 있어 텍스트 중복 제거).

### 1-3. 히어로
- 블러 처리된 **한옥 지붕·산 능선 SVG 배경**(자체 제작, 브랜드 청록톤).
- 카피: "데이터가 찾은 쇠퇴 소도시, 머물수록 살아나는 여행".

### 1-4. 임팩트 지표 줄 (`.stats`)
- **12 숨은 소도시 · 240 관광 명소 · 1,546 문화유산 · 5종 TourAPI 서비스**.

### 1-5. 카드 레이아웃 (5개 → 대표1 + 4)

## 2. 하루코스 (map/course.html)

기준 = 숨은 매력지(momuri.html)의 물방울 핀·팝업 방식으로 통일.

- **마커**: 번호 원(`.nm`) → **반투명 번호 물방울 핀**(`numPin(color,n)`, opacity .85, 팁이 좌표를 가리킴).
- **클릭 팝업**(`stopPopup`): 사진 + "N. 분류" 배지 + 이름 + 주소 + **[구석구석][네이버 지도]** 링크(`extLinks`). `mkPopup`으로 z-index 최상단 강제.
- **타임라인 장소 클릭** → 지도 이동(flyTo) + 해당 마커 팝업 토글.
- **범례**: 지도 좌하단 `🔵 명소 · 🟡 맛집`(`.legend`). 색은 course의 기존 스킴 유지(명소=파랑 #2e79c4, 맛집=주황 #f59e0b — 타임라인 배지와 일치).
- 추가 헬퍼: `numPin / mkPopup / https / extLinks / stopPopup`, `#map{position:relative}`.

## 3. 체류 (map/stay.html)

### 3-1. 정착 대시보드 버튼 깨짐 (8-2) — 이미 수정됨
- `.settle`을 **2×2 균등 그리드 + `word-break:keep-all`**로(커밋 dd71731). 격리 렌더로 4버튼 한 줄씩 깔끔 확인. **라이브에도 반영됨**.
- 사용자가 "아직 깨진다"고 본 건 **브라우저 캐시**(Ctrl+F5로 해결). 색은 청록(`--brand-d`)이며 파랑 아님.

### 3-2. 워케이션 아이콘 범례
- 주간 캘린더가 평일 💻 / 토 🚶 / 일 ☕만 표시하고 설명이 없어 "💻가 뭔지 모름" → **범례 추가**: `💻 워케이션(원격근무) · 🚶 나들이 · ☕ 휴식` (`calHTML()` 하단, `.cal-legend`).

### 3-3. 채용 패널 — "공기업만 나온다" 오해 해소
- 원인: 카드로 나열되는 공고는 **잡알리오(공공기관 공식 API)** 뿐. 민간·중소기업은 **고용24 링크**로만 연결돼 카드에 안 보임.
- 조치: 라벨/안내 정리 — **민간·중소기업 채용(고용24)** vs **공공기관 채용(잡알리오)** 구분 명시. 카드=공공기관 공식 API임을 안내.
- 🔜 **미해결(다음 작업)**: 민간 공고를 카드로 나열하려면 **고용24 채용 API 연동** 필요(현재 딥링크만).

### 3-4. 장기 숙소 지도 레이어 ⭐신규
- 🏡 장기 숙소 패널에 **토글 버튼**: "🗺 지도에 숙소 N곳 표시" ↔ "숨기기"(`.staytoggle`, `toggleStays()`).
- **보라색 🏠 반투명 물방울 핀**(`stayPin()`, #7c5cff) + 클릭 **팝업**(`stayPopup`: 사진·숙소명·주소·**[네이버 지도][구석구석]**).
- **별도 마커 배열** `stayMarkers`로 관리 → 월/일 모드·주차·일자 전환에도 유지(각 `drawWeekMap`/`drawMap` 끝에서 `drawStays()` 재호출).
- 전역: `stayMarkers, staysOn, curRegion, curStays`. `render()`에서 `curStays = (region.stays||[]).filter(좌표 있는 것만)`.
- 좌표 있는 숙소만 마커화(상주·함안=숙소 0곳 → 버튼 미표시).
- **마트·병원·로컬일자리는 좌표 데이터가 없어 지도 레이어 불가** → 정착 대시보드/채용 패널의 네이버·고용24 링크로 유지가 현실적.

## 4. 검증 / 한계 (중요)

- 홈(index.html)·격리 UI(정착 버튼·로고·배지)는 **Chrome 헤드리스 스크린샷으로 육안 검증** 완료(데스크톱/모바일).
  - ⚠️ 헤드리스 **최소 창폭 ~500px** 아티팩트: 430px로 찍으면 500px 레이아웃을 잘라 찍어 "잘린 것처럼" 보임 → 실제 모바일과 다름. ≥500px로 검증할 것.
- **지도 페이지(course/stay)의 마커·팝업은 로컬 육안 검증 불가**: `routogl.RoutoStyle`(Routo 지도)의 키가 **배포 도메인 전용**이라 localhost/file에선 타일·마커가 안 뜸. 부팅도 `map.on('load')` 안이라 데이터/드롭다운도 안 채워짐.
  - 대신: **JS 구문검사(`node --check`) 통과**, 범례 등 정적 요소는 확인, 마커/팝업은 momuri의 검증된 패턴을 그대로 이식.
  - ✅ **실사이트에서 최종 확인 필요**(§6 테스트 절차).
- 로컬 지도 미리보기: `cd map && python -m http.server 8137` → http://localhost:8137/... (단, Routo 키 문제로 지도는 회색).

## 8. 기술 메모 (이어서 작업할 때)

- **마커/팝업 패턴 통일**: `numPin(color,n)`(번호 물방울), `stayPin()`(🏠), `mkPopup(opts,html)`(z-index 강제), `extLinks(name)`/`stopPopup`/`stayPopup`. momuri·course·stay 모두 동일 패턴.
- **색상**: 관광/명소 파랑 `#2e79c4`, 맛집 주황 `#f59e0b`, 숙소 보라 `#7c5cff`(KCOL). 홈 브랜드 청록 `#14b8a6`/`#0f766e`.
- **데이터**: `map/data/momuri.json`(지역12, 각 spots/foods/cafes/culture/shopping/leports/stays/festivals + heritage). stays 필드=`title,addr,lat,lng,img,cat3`(lat/lng는 문자열 → `+s.lng`). `courses.json`(하루/다일 코스), `jobs.json`(잡알리오).
- **이미지**: TourAPI `tong.visitkorea.or.kr` (https 지원 확인). http면 mixed-content로 막히니 `https()`로 승격.
- **배포**: `git push` → Cloudflare 자동 재배포. HTML은 `/map/xxx.html`→`/map/xxx` 307 리다이렉트.
