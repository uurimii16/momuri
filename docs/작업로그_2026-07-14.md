### 🅱 지도 스택 Routo → MapLibre GL 전면 이식 (커밋 `59ac518`·`b450929`·`8812979`)
사용자가 **tourgo 지도(MapLibre+VWorld)의 테크/비주얼을 훨씬 선호** → 4개 지도 페이지 전부 이식.
- **Routo GL은 Mapbox GL 포크라 API가 거의 동일** → `routogl.Map/Marker/Popup` → `maplibregl.*` 로 기계적 치환.
- **베이스맵**: VWorld(국토교통부) 기본 + 실패 시 **Carto 무키 폴백 자동 전환**(tourgo와 동일 `BASE_STYLE`).
  - `VWORLD_KEY`는 tourgo와 공용. Referer 검증: **localhost·tourgo 도메인은 등록됨, momuri 도메인은 미등록** → live에선 Carto 폴백으로 뜸. (원하면 브이월드 포털에서 `momuri.uurimii16.workers.dev` 등록 시 VWorld로 바뀜.)
- 내비게이션/스케일 컨트롤 추가.

#### ⚠️ 이식 중 겪은 하드 트러블슈팅 (재발 방지용 — 중요)
1. **MapLibre는 `addSource`/`addLayer` 전에 스타일 로드 완료가 필수**(Routo는 관대했음). → `onMapReady`는 `map.once('load')` 게이트 사용.
2. **애니메이션 카메라(`fitBounds`/`flyTo` with `duration>0`)는 `requestAnimationFrame` 의존 → 백그라운드(hidden) 탭에서 rAF가 정지되어 카메라가 목표 위치로 이동조차 못 함.** → **전부 instant(`duration:0`)로 통일**(rAF 불필요, 모든 탭 안정). 실제 포그라운드 사용자는 애니메이션도 정상이지만 안정성 위해 instant 채택.
3. **60fps rAF 리페인트 펌프는 렌더러를 프리즈시킴 → 절대 금지.** 대신 `data`/`moveend` 이벤트 리페인트 + 로드 직후 `setInterval` 가벼운 kick 사용.
4. **자동화 브라우저(claude-in-chrome) 탭은 항상 hidden → MapLibre `load`·렌더가 throttle되어 캔버스/데이터 검증이 어려움**(실사용자 포그라운드는 정상). 검증은 지도 **클릭(상호작용)으로 렌더를 깨워** 확인. course.html은 클릭 후 상주 코스·마커5·경로선 완전 렌더 확인함.
5. **반복 리로드는 WebGL 컨텍스트 고갈**(Chrome ~16개) → 검증 시 **새 탭 필요**.

## 2. 고용24 채용 API — 계정 제약으로 불가 (중요)
- **고용24/워크넷 채용정보 API는 "기업회원 전용"** → 개인 data.go.kr 키로 접근 불가. (`analysis/build_jobs.py` 주석에 명시.)
- 그래서 기존에 **잡알리오(기획재정부 공공기관 채용, data.go.kr 1051000)**로 대체하고 민간은 고용24 딥링크로 처리한 것.
- **코딩으로 우회 불가**한 계정 문제. 사업자 등록 → 기업회원 계정 신청해야 민간 채용 카드화 가능.
- 현재 방식(잡알리오 공공기관 카드 + 민간 딥링크)이 개인 계정으로 가능한 최선.
