머무리 숨은 매력지 쇠퇴지수 지도 지표 방법론

복합 관광쇠퇴지수 — 지표 설정 및 계산 방법

지역 관광의 쇠퇴는 "방문객이 적은 곳"이 아니라, 방문이 줄고 · 매력이 약해지고 · 동선에서 소외되는 다차원 현상이다. 본 지수는 공개 데이터만으로 이 셋을 정량화해 전국 시·군을 진단한다. 이 지수 산출 자체가 데이터 분석의 핵심 결과물이다.

1. 데이터 3소스

축원재료(공식 데이터)출처
① 방문외지인 연간 방문자수 (2019·2022·2024)한국관광공사 지역별 방문자수 API (data.go.kr 15101972)
② 매력재방문의향·추천의향 (1~5점 설문)국민여행조사 원자료
③ 연결여행 내 방문 순서 (A→B→C)국민여행조사 원자료
자산국가유산 지정 15,728건 좌표국가유산청 검색 API
원재료(방문수·설문응답·유산목록)는 공식에서 받지만, 아래 "지수"는 전부 우리가 통계로 산출한 것이다. 이 지수가 서비스의 차별점(moat)이다.

2. 세 축의 계산

① 방문 쇠퇴 ax_visit

연도별 외지인 방문자수로 회복률과 최근추세를 구해 표준화한다.

회복률 = v2024 / v2019, 추세 = v2024 / v2022
ax_visit = z(−회복률) + z(−추세) (방문 줄수록 ↑)

② 매력 쇠퇴 ax_attract

재방문의향(A10)·추천의향(A11)을 여행1차 방문지에 귀속하고 표본가중치(WT_DOM)로 가중평균 후 표준화. 표본 30건 이상 지역만.

ax_attract = z(−재방문의향) + z(−추천의향) (만족 낮을수록 ↑)

③ 연결 소외 ax_connect

여행 내 연속 방문 A→B를 방향 엣지, 지역을 노드로 하는 네트워크를 WT_DOM 가중으로 만들고, 연결중심성(degree centrality)을 구한다.

연결중심성 = 연결된 이웃 수 / (전체 노드−1)
ax_connect = z(−연결중심성) (동선 고립일수록 ↑)

3. 복합 지수

복합쇠퇴지수 = ( ax_visit + ax_attract + ax_connect ) / 3

세 축을 등가중 평균(가중치 임의성 배제). 표준화 기준 모집단은 도지역 시·군(광역시 자치구·세종 제외) + 표본 30↑ + 3축 존재 = 123~133곳.

실제 산출 예시

지역ax_visitax_attractax_connect복합쇠퇴 성격
충북 진천군−0.47+4.70+0.981.74매력쇠퇴 압도(재방문↓)
경북 상주시−0.60+4.16+1.261.61매력↓ + 동선 소외
경기 여주시+3.33+1.01+0.281.54방문쇠퇴 압도

같은 "쇠퇴 상위"라도 왜 쇠퇴인지가 다르다 — 이것이 지역별 맞춤 처방의 근거가 된다.

4. "숨은 매력지" 도출

쇠퇴지수를 국가유산 밀도(시군구 집계)와 교차한다. 문화유산은 소수 지역에 몰린 편포라 log 변환 후 표준화한다.

숨은매력지 점수 = z(쇠퇴) + z(문화유산 자산)
세그: 쇠퇴>중위 AND 자산>중위 → "숨은 매력지(문화유산형)"

쇠퇴했지만 살릴 자원(문화유산)이 있고, 동선에서 소외된 소도시를 선별한다. 자산 없는 극단 쇠퇴지(접근성 극악·자원 희소)는 제외한다. 인구 50만+ 대도시(지방자치법 대도시 기준)도 타깃에서 제외한다.

5. 사용한 알고리즘

  • z-표준화 (x−평균)/표준편차 — 서로 다른 단위를 비교 가능하게
  • 표본가중평균(WT_DOM) — 설문을 전국 규모로 복원
  • 방향 네트워크 + degree centrality — 동선 허브·소외 측정 (networkx)
  • 등가중 합성 — 3축 단순평균
  • log 변환 — 편포(문화유산) 완화
  • 분위(quantile) 5단계 — 단계구분도 구간화

6. 한계

  • 국민여행조사 단일연도(구조 중심) → 방문추세는 방문자수 API로 보강
  • 방문자수(통신데이터)는 일 단위 중복 정의; 행정표준코드↔설문코드 이름 매칭
  • 통합시(구 분할)·일부 시군구 매칭 한계 (경계 123/133 조인)
  • 인구 50만+ 제외로 대도시의 진성 매력쇠퇴 신호 일부 손실 가능
