## 3. 현재 상태 (2026-07-06)
### 완성된 것
- **분석 파이프라인** 완성: 쇠퇴지수 → 대도시 필터 → 숨은매력지 후보 → TourAPI 콘텐츠 → choropleth(Jenks).
- **3페이지 웹사이트** (`map/`):
  - `momuri.html` — 숨은매력지 12개 지도 데모(메인)
  - `decline-map.html` — 시군구 쇠퇴지수 단계구분도(Jenks 자연분류, 실제 지수 범위 범례, 하단 설명)
  - `methodology.html` — 지표 방법론 설명서
  - 밝은 테마 + 머무리 로고(인라인 SVG) + 한국관광공사 TourAPI 출처 표시.
- **루트 랜딩** `index.html`(3페이지 진입) + 페이지 간 내비게이션.
- **✅ Netlify 배포 완료** — GitHub 연동, `netlify.toml`(publish=루트). `git push` 하면 **자동 재배포**. 지도(Routo)도 공개에서 정상 표시 확인.
- **README.md 웹앱 기준으로 재정리** + `requirements.txt`(재현용) 추가. 다른 컴퓨터에서 클론만 하면 웹 즉시 작동.

### 실행 / 배포
```bash
# 로컬 미리보기
cd map && python -m http.server 8901   # http://localhost:8901/momuri.html (포스터는 8899)

# 배포 갱신 (Netlify 자동)
git push        # → 1분 뒤 https://momuri.netlify.app/ 자동 반영
```

## 4. 저장소 구조
```
├─ index.html             배포용 루트 랜딩(3페이지 진입)
├─ netlify.toml           Netlify 배포 설정(publish=루트)
├─ README.md              웹앱 셋업·재현 가이드(정본)
├─ requirements.txt       분석 재현용 파이썬 패키지
├─ 작업기록_인수인계.md      ← 이 파일
├─ docs/
│  ├─ 웹앱_기획_과제4.md     핵심 기획서 (개념·3층 구조·전략)
│  ├─ 보고서_골격.md / 보고서_초안.md / 인용자료_정리.md   (포스터 자산)
├─ figures/               분석 그림 4종
├─ map/                   웹사이트 (momuri/decline-map/methodology.html + data/ + 구 app.js·style.css)
└─ analysis/              분석 코드 + 산출물 (build_*.py, pull_*.py, tourapi_fetch.py)
```
> 제외(.gitignore): `.sav .xlsx .pdf .zip`, `map/data/sigungu.geojson`(18MB 경계, 재다운로드 가능). 웹 작동엔 불필요(계산된 산출물은 커밋됨).

## 5. 데이터 파이프라인 (재현 순서)
```bash
pip install -r requirements.txt   # pandas numpy pyreadstat networkx
# 환경변수: TOURAPI_KEY(필수), DATAGO_KEY(방문자수)
python analysis/pull_visits.py        # ① 방문 → visits.json
python analysis/build_index.py        # ②매력·③연결 (knts2025.sav 필요) → index_axes.json
python analysis/combine_index.py      # 복합쇠퇴지수 → index_full.json
python analysis/build_candidates.py   # 숨은매력지 후보(대도시 필터) → candidates.json
python analysis/build_choropleth.py   # 단계구분도(Jenks) (sigungu.geojson 필요)
python analysis/build_service_data.py 12   # TourAPI 콘텐츠 → map/data/momuri.json
```
※ 산출물이 이미 커밋돼 있어 웹만 볼 거면 생략 가능.
